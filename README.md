# Scarab

Scarab is a lightweight web routing framework built on Sinatra, similar to
Sinatra's "namespace" plugin.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'scarab', git: 'https://github.com/Co-Chu/Scarab'
```

And then execute:

    $ bundle
